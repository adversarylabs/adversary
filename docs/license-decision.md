# License decision

Status: owner decision required.

On 2026-07-11 the repository contained no license file and GitHub reported no
detected license. There is not enough authoritative evidence to choose a
license or assert copyright ownership on an owner's behalf. Until an owner or
authorized legal representative selects a license, `LICENSE` records that no
license grant is made and the Homebrew formula deliberately omits `license`.

Resolution criterion: an owner-approved pull request must name the license,
add its complete canonical text, identify any required notices, update package
metadata, and document whether prior contributions are covered. This is the
explicit repository decision for CLI-023 recommendation 9.
